# Optimization-Algorithms-Library
This is a research-oriented library implementing several optimization algorithms for nonlinear problems. All codes in this repository are written in **MATLAB**.

The numerical problems considered in this project mainly fall into two categories:
1. **Problems involving numerical integration**
2. **Large-scale derivative-free optimization problems**


To solve these problems, we implement and study several classes of optimization algorithms, including:
* **Gradient-based methods**
* **Nonlinear Conjugate Gradient methods**
* **L-BFGS methods**
* **Trust-region methods** (to be released)
